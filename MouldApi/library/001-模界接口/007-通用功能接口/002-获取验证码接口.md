### 请求地址

### 请求类型

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- || --- |
| phone | string | 是 | 手机号 || -- |

```javascript
{

  "data":"6915"
}
```
### 返回错误JSON示例

}
