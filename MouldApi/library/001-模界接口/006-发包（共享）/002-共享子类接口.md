# 发包分类列表接口
>维护人员：**奎神**  
>创建时间：2016-06-12

## 接口简介
--包括筛选，搜索

## 接口详情

### 请求地址
mw/index.php?g=api&m=product&a=getSubclass&id=

### 请求类型
GET

### 请求参数
| 参数名 | 类型   | 必填 | 描述   | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- |---|
| id | string | 是   | 父类id ||1|
### 返回正确JSON示例
```javascript
{
  "status": "1",
  "data": [
    {
      "id": "1",
      "type_name": "分类名称",
      "type_id": "父级id",
      "imgurl": "待定字段"
    }
  ]
}
```
### 返回错误JSON示例
```javascript
{
  "status":"0",
  "data":"错误信息"
}
```
